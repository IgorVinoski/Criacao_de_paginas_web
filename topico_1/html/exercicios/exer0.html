Listagem das principais tags da HTML:

  • <head>

    O elemento <head> gerencia os metadados da página, além disso, é através dele que é informado o título e links para scripts e arquivos CSS.

  • <style>

    O <style> gerencia as informações de estilo na página, essas estilizações fazem parte do CSS. A tag <style> pode ser utilizada no próprio HTML, assim, não necessitando de um novo arquivo para gerenciar o CSS.

  • <title>

    O <title> define o título do documento. Essa tag é responsáel pelo título mostrado na barra de pesquisa do navegador.

  • <body>

    O <body> gerencia o conteúdo do corpo de uma página. Só é permitido um <body> por documento.

  • <footer>

    O <footer> gerencia o rodapé do documento. Normalmente, o conteúdo final da página é definido como rodapé, que tem as informações armazenadas no <footer>.

  • <label>

    O <label> representa uma legenda a um item HTML. Antes da introdução de elementos na página, pode existir um <label> para informar o que aquela informação significa.

  • <i>

    O <i> normalmente é usado para definir o itálico em determinada parte do texto. É utilizado para destacar algo do restante do texto.

  • <strong>

    O <strong> normalmente é usado para definir o negrito em determinada parte do texto. É utilizado para destacar algo do restante do texto.

  • <small>

    O <small> normalmente é utilizado para definir uma subscrição, deixar uma parte do texto menor. É utilizado para destacar algo do restante do texto.

  • <br>

    O <br> produz uma quebra de linha.

  • <hr>

    O <hr> representa um espaço entre os parágrafos. Utilizado para diferenciar estrofes de uma música e coisas do gênero.

  • <b>

    O <b> representa uma distinção do texto para algo destacado com a tag. Normalmente é utilizado para destacar palavras-chaves e conteúdos importantes do texto.

  • <div>

    O <div> é um container genérico. Ele pode ser utilizado para agrupar diversos elementos, que podem serem estiliazados através do CSS informando os ids ou class da div. Deve ser utilizado quando não tem outro elemento próprio do HTML para executar a função.

  • <form>

    O <form> representa um seção do documento para interatividades. Utilizado antes de elementos de inserção de dados na página. O elemento de diversos tipos, que reprentam o dado que é informado pelo usuário.

  • <input>

    O <input> é usado para criar campos de inserção de dados.

  • <tr>

    O <tr> define as linhas de uma tabela.

  • <td>

    O <td> define as colunas de uma tabela.

  • <span>

    O <span> é um container genérico fraseado. Pode ser utilizado para agrupar elementos para a estilização ou elementos que compartilham dos mesmos atributos. Deve ser utilizado somente quando nenhum outro elemento for apropriado. É muito parecido com o elemento div, porém, <span> é um elemento em linha enquanto div é um elemento de nível.

  • <img>

    O <img> é utilizado par aa inserção de imagme no documento.

  • <audio>

    O <audio> é utilizado para atribuir um áudio no documento.

  • <canvas>

    O <canvas> pode ser utilizado para desenhar elementos gráficos com scripts.

  • <datalist>

    O <datalist> contém um conjuntos de elementos pré definitos.

  • <details>

    O <details> é utilizado para criar um campo onde o usuário pode obter mais informações.

  • <embed>

    O <embed> incorporta conteúdo externo ao documento. É utilizado para adicionar um vídeo, por exemplo.

  • <ul>

    O elemento <ul> é utilizado para definir uma lista desordenada. Ou seja, uma coleção de itens sem ordem numérica pré definida com posiçõe irrelevantes.

  • <ol>

    O elemento <ol> é utilizado para definir uma lista de itens ordenados. Os elementos dessa lista são mostrados com uma contagem que os precede.

  • <li>

    O elemento <li> é utilizado para definir um item que faz parte de uma lista.
